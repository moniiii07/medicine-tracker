# 💊 Medicine Tracker

A medicine reminder and drug interaction checker built with Python and Streamlit.

# Live Demo
🔗 [Open App](https://medicine-tracker-t2hczu45otrgmgcgg2nzuz.streamlit.app)

## Features
- Add and manage medicines with dosage and timing
- Today's schedule — see which medicines are due now
- Drug interaction checker using FDA's official database
- Clean dark UI with medicine cards

## Tech Stack
- Python
- Streamlit
- SQLite (local) / PostgreSQL via Supabase (cloud)
- OpenFDA Drug Label API

## Disclaimer
This tool is for informational purposes only.
Always consult a doctor or pharmacist before making medication decisions.
